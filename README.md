# GamerScraper
this is a simple web scraper made in C#

## **Usage** 

There are 3 options:
- downloadpage
- downloadvideo
- downloadfile

for every option you need to specify a URL and a path with the 
- -u / --URL flag.
- -p / --path flag.

## **Notes**
***You can only download Youtube videos with the downloadvideo mode.***
